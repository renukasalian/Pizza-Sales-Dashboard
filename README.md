# Pizza-Sales-Dashboard
1. Pizza Sales Insights Dashboard:
  A modern, interactive Power BI dashboard built to analyze pizza sales performance through dynamic KPIs, category-wise breakdowns, and detailed best/worst seller insights.

2. Short Description / Purpose:
  The Pizza Sales Dashboard is an analytical and visually engaging Power BI report designed to uncover key business insights from pizza sales data. It helps business track revenue, order trends, product         performance, and customer behavior patterns through intuitive KPIs and interactive visuals. The dashboard provides a complete year-round analysis to support decision-making in marketing, operations, and product strategy.

4. Tech Stack:
  This dashboard was built using the following tools and technologies:
  - Power BI Desktop
  - Power Query (data cleaning & transformations)
  - DAX (Data Analysis Expressions) for calculated KPIs and measures

5. Data Source:
  The dataset consists of 48,620 rows and 17 columns, containing detailed transactional information for pizza orders.
  The data was cleaned and transformed using Power Query, and all KPIs were calculated using DAX measures.

4. Features:
  Business Problem:
  The company wants to understand its pizza sales performance across an entire year.
  Leadership needs clarity on:
  Which pizzas generate the highest revenue?
  What categories and sizes perform best?
  Which products need attention due to poor performance?
  What are customer ordering patterns across days & months?
  What is the overall business health in terms of revenue, orders, and sales volume?

5. Project Goal:
  To build an interactive Power BI dashboard that delivers clear insights into:
   - Revenue trends
   - Best-selling vs worst-selling pizzas
   - Category-wise and size-wise contributions
   - Daily and monthly sales patterns
   - Key business performance indicators

6. Dashboard Highlights / Key Features:
  KPI Metrics:
  The report includes dynamically calculated KPIs:
  - Total Revenue
  - Average Order Value
  - Total Orders
  - Average Pizzas Per Order
  - Total Pizza Sold

  Interactive Filters:
  - Category filter
  - Date range slicer
  - Navigation buttons for multi-page report

  Pages Included:
    1. Home Page
       - KPI's
       - Daily Trends for Total Orders
       - Monthly Order Trends
       - Sales % by Pizza Category
       - Revenue % by Pizza Size
       - Total Pizza Sold by Category
       - Overall insights
       - Slicers
       - Date range slider
    2. Best/Worst Selleres Page
       - Top 5 Pizza by Revenue
       - Top 5 Pizza Quantity Sold
       - Top 5 by Total Orders
       - Bottom 5 Pizzas by Revenue
       - Bottom 5 by Quantity
       - Bottom 5 by Total Orders
       - Overall Insights
       - Slicers
       - Date range slider
